###  Getting Started

1. **Clone the Repository**
```bash
git clone https://github.com/yourusername/kaiju-tower-defense.git
cd kaiju-tower-defense
```

2. **Install Dependencies**
```bash
pip install -r requirements.txt
```

3. **Launch the Game**
```bash
python main.py
```
